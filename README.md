# 🚀 Autonomous CTO Agent

AI-powered Autonomous CTO Agent for repository analysis using LangGraph, RAG, Qdrant, FastAPI, React and Gemini.

## Tech Stack

- FastAPI
- React
- LangGraph
- Gemini
- Qdrant
- Docker
- PostgreSQL

## Features

- Repository Analysis
- Architecture Visualization
- Repository Health Score
- File Tree Explorer
- Codebase Q&A using RAG
- GitHub Repository Ingestion
